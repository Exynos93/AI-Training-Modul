# AI-Powered Data Cleaner

## Overview
AI-Powered Data Cleaner is a Python-based tool designed to help data scientists and AI practitioners preprocess datasets efficiently. It automates the handling of missing values, outliers, normalization, and encoding, making data ready for machine learning models.

## Features
- Handle missing values with various strategies (mean, median, mode).
- Detect and handle outliers using statistical methods.
- Normalize numerical features to a standard scale.
- Encode categorical features automatically.
