Objective: Participants are asked to clean a provided dataset by handling missing values, scaling features, and encoding categorical variables.
Instructions:
Load the dataset.
Identify and handle missing values.
Standardize numerical features.
Encode categorical features using one-hot encoding.
Submission: Submit a cleaned version of the dataset as a CSV file.

notebooks/Intro_to_ML.ipynb:
A Jupyter notebook that walks participants through:

Introduction: Explaining the concept of supervised learning.
Loading Data: How to load and explore datasets.
Data Preprocessing: Step-by-step guide on handling missing values, scaling, and encoding.
Model Building: Implementing a Linear Regression and Decision Tree model.
Model Evaluation: Evaluating model performance using appropriate metrics.
